colores en la web

color--> es un modulo de css propiedades

  • color>>le da color al texto
  • background-color>> le da color al fondo
  • currentcolor>>se utilizo para la primera variable que se utilizo em css, ya que representa el color actual

valores

  • color keywords>> es una palabra que representa al color, ejm: red, green, etc. tambien esta considerando como color el valor de transparent
  • nodo de color RGBrgb(r=[0-255], g=[0-255], b=[0-255])
    • tres canales: rojo(r), verde(g), azul(b) los tres canales combinados dan muchos colores
    • los valores son del 0 al 255, por que 2^8=256, de esta manera el total de combinados seria: 2^(8*3)=16.777.216

    los tres el valor en 255 dan el color blanco

    los tres el valor en valor 0 dan el color negro

    si los tres son iguales = dan un color gris

    rgba()>>a:se refiere a la transparencia, sus valores estan en el rango de 0 a 1

  • Notacion hexadecimal>>no es otro, sigue siendo RGB,pero ya no se invoca la funcion rgb(), se usa la funcion hexadecimal por ser de 16 en 16, por lo que tiene los valores de 0 a 16[0-F], es solamente otra forma de denotar el modo RGB
  • modo de color HGL y HLSAes el modo de color mas recomendable, significa:
    • H >>tonalidad del color, hace referencia al circulo cromatico es decir partiendo del RGB seria: 0 grados= red, 120 grados= green, 240 grados=blue
    • S >>saturacion>> intensidad del color, sus valores son porcentuales, donde 0%= gris y 100%= color puro
    • L >>luminosidad >> es el brillo del color, donde 0%= negro, 50%=color puro, 100%=blanco

      • De esta forma tendriamos hls(grados, porcentaje, porcentaje)
