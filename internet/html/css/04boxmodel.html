Box Model o Modelo de caja

El box model es el algoritmo a traves de cual el navegador dibuja las cajas en la pantala, solo se aplica a los elementos de Bloque, cuenta con tres elementos

  1. Contenido-->content-bos
  2. Separacion entre el contenidoy el borde-->pading-box
  3. Border-->border-box

Tambien es necesario mencionar el margen que es una propiedad externa e indica la separacion entre cajas, la propiedad es el margin, ejemplo
