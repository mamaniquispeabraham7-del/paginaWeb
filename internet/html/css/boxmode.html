Elemenetos de linea(inline) y bloque(block)

Caracteristicas inline

  1. No crean nuevas lineas para cada elemento
  2. No se puede definir el ancho ni el alto
  3. El ancho se ajusta de acuerdo al contenido

Caracteristicas block

  1. Crean una linea para cada elemento
  2. Ocupan todo el espacio horizontal disponible
  3. tienen ancho y alto
