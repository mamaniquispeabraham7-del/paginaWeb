selectores

selectores sencillos

1.Selctor universal

Selecciona todos los elementos del documento web, se representa por *

2.Selectores de tipo o elemento

Es el selector mas sencillo, ya que simplemente es una etiqueta de html

Selector de clase

Una clase es un atributo de un elemento u etiqueta de html o xml.Este es el mas recomendable para dar estilo ya que puede ser reutilizado

Selector por id

Un id es un atributo de un elemento u etiqueta de html o xml. Su uso es mas recomendable para JS, ya que no puede ser reutilizado en css

Selectores compuestos

Para este tipo de selectores, hay mas de un selector combinado

Selectores agrupados

Todos los selectores estan agrupados por el caracter ,

SELECTORES DESCENDIENTES

Indican cuando un elemento es hijo, nieto, etc. Se agrupan con un espacioejemplo
